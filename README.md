# Project Niche-Layer (PNL)
### 「道路」というレガシーOSを、3次元でリファクタリングする。

**Project Niche-Layer (PNL)** は、既存の2次元道路インフラの限界（技術負債）を打破し、上空50mの巡航レイヤーと「ポッド換装式」の移動プロトコルによって、都市空間を再定義するオープンソース・プロジェクトです。

## 🛠 技術的コア・コンセプト
本プロジェクトは車両の開発ではなく、**「移動の実行基盤（プロトコル）」**のリファクタリングを提案します。

1. **疎結合アーキテクチャ:** 「乗員ポッド」と「推進フレーム」を分離。
2. **非同期バケツリレー:** 10階建て屋上ハブでの迅速なフレーム交換（ホットスワップ）。
3. **マルチレイヤー都市OS:** 都市をレイヤー0（人間）、レイヤー2（移動）、レイヤー-1（地下）に分離。
4. **多重化フェイルセーフ:** 逆噴射システム、パラシュート、衝撃吸収構造による極めて高い安全性の確保。
