# PNL Simulator (Unity)

## 📌 概要
Project Niche-Layer の「ポッドとフレームの疎結合」および「3D飛行挙動」を物理演算で検証するためのシミュレータです。

## 🛠 技術要件
- **Unity**: 2022.3 LTS 以上推奨
- **Language**: C#
- **Physics**: Built-in Physics または Havok (検討中)

## 📋 開発タスク（Issue）
- [ ] [Issue #1] Pod/Frame 物理APIのC#インターフェース実装
- [ ] [Issue #1] ローター N+1 冗長化の物理挙動シミュレーション

## 🧪 主要パラメータ
- ポッド空車重量: 400kg
- 最大積載量: 250kg
- フレーム自重: 350kg
